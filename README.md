# 💰 Expense Tracker

A modern and interactive expense tracking web app built using pure HTML, CSS, and JavaScript.  
It helps users manage income, track expenses, and gain insights through real-time analytics and visualizations.

---

## 🚀 Live Demo
👉 https://prakharojha960.github.io/expense-tracker/

---

## ✨ Features

### 📌 Core Functionality
- Add income & expense transactions  
- Edit existing transactions  
- Delete transactions  
- Real-time balance calculation  
- Automatic income vs expense tracking  
- Persistent data using LocalStorage  

---

### 📊 Analytics & Insights
- Category-wise expense breakdown  
- Dynamic pie chart visualization (Canvas API)  
- Smart spending alerts:
  - ⚠️ Overspending detection  
  - ⚠️ High spending (>80% of income)  
- Live financial summary (Income, Expense, Balance)  

---

### 🎯 UI/UX Highlights
- Clean, modern dashboard layout  
- Color-coded transactions (income vs expense)  
- Smooth real-time updates (no page reload)  
- Minimal and intuitive interface  

---

## 🧠 Highlights

## ⚙️ How It Works

1. Add a transaction (amount, type, category, date)  
2. Data is stored locally in the browser  
3. UI updates instantly:
   - Balance  
   - Income  
   - Expense  
   - Category breakdown  
4. Chart updates dynamically based on expenses  
5. Alerts trigger based on spending patterns
